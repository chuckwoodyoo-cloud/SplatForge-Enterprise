# SplatForge Enterprise

基于开源 SuperSplat 二次开发的企业级三维高斯点云编辑、清理与交付工作台。

本项目面向 3D Gaussian Splatting 资产的查看、编辑、清理、格式转换、视频输出和企业化交付需求，适用于数字孪生、空间扫描、文旅展示、工业可视化、影视预演、房地产展示、教育培训等场景。

## 项目定位

SplatForge Enterprise 的目标是把开源 SuperSplat 的浏览器端高斯点云编辑能力，扩展为更适合企业资产处理流程的三维工作台。

核心目标：

1. 降低三维高斯点云资产查看、清理和交付门槛。
2. 强化 SPZ 等轻量格式的导入、导出和展示体验。
3. 提供面向真实项目的端部精修、区域复制、填充选择和画笔清理能力。
4. 支持图片渲染、视频渲染和实时视口录制，便于汇报与交付。
5. 为后续企业资产库、权限审计、私有化部署和平台集成预留扩展空间。

## 主要功能

### 多格式资产导入与导出

- 支持 PLY、Compressed PLY、SPLAT、SPZ、SOG 等格式。
- 支持 SPZ v4 与 legacy v3 导出选项。
- 支持导出 Viewer 包，便于客户查看和项目交付。
- 支持多素材导入、变换、组合和整理。

### 高质量浏览器端查看

- 基于 PlayCanvas Engine 的浏览器端三维渲染。
- 支持飞行模式、环绕模式、视野角、背景色、色调映射和球谐阶数调节。
- 支持视口质量设置，提高素材细节显示效果。
- 支持网格、边界、相机姿态、中心点和选区轮廓等辅助显示。
- SPZ 导入后网格可根据当前可见点云自动估算地面高度，减少网格悬空或穿模。

### 点云编辑与清理

- 支持矩形、套索、多边形、画笔、填充、盒选、球选等选择方式。
- 支持选择、反选、锁定、解锁、删除、拆分、复制等常用操作。
- 优化填充选择和画笔操作，降低误删范围过大的风险。
- 支持操作撤销与重做。

### 端部精修

新增“端部精修”工具，用于处理高斯点云头尾区域常见的糊边、漂浮点、大尺度点和低透明度噪点。

能力包括：

- 自动检测低透明度、大尺度和大体积点。
- 仅作用于当前框选或画笔选区。
- 支持轻度、中度、强力三档清理。
- 支持只清前端、后端或当前视角深度层。
- 删除前高亮预览，确认后再执行。

### 区域复制到目标位置

新增“复制到目标”能力，可将当前选中的点云区域复制到用户点击的目标位置，适用于局部补洞、结构复用、快速占位和演示修复。

### 图片、视频与实时录制

- 支持当前视图图片导出。
- 支持离线视频渲染，可配置分辨率、格式、编码器、帧率、帧范围和码率。
- 支持实时视口录制，直接记录浏览器中的漫游效果。
- 未导入素材时不会启动录制，录制停止后会停止相机交互，避免镜头继续运动。

## 环境要求

推荐环境：

- Node.js >= 20.19.0
- npm
- Chromium 内核浏览器，用于获得更好的 WebCodecs 视频录制和渲染兼容性

## 本地开发

安装依赖：

```sh
npm install
```

启动开发模式：

```sh
npm run develop
```

然后访问：

```text
http://localhost:3000
```

开发模式会监听源码变化并自动重新构建。浏览器端如果存在缓存，请使用强制刷新，或在开发者工具中禁用缓存。

## 常用命令

```sh
# 构建生产版本
npm run build

# 启动静态服务
npm run serve

# 开发模式：监听构建 + 本地服务
npm run develop

# 代码检查
npm run lint
```

## 使用流程

典型工作流：

1. 打开应用并导入 PLY、SPZ、SPLAT 或其他高斯点云资产。
2. 调整视口质量、相机模式、色调映射和辅助显示。
3. 使用框选、画笔、填充等工具选择需要处理的区域。
4. 通过删除、拆分、端部精修或区域复制完成局部清理与修复。
5. 根据交付需求导出 PLY、SPZ、SOG、Viewer、图片或视频。
6. 将交付包用于客户汇报、项目验收、线上展示或内部归档。

## 部署建议

可根据使用场景选择不同部署方式：

1. 本地演示部署：适合单机试用、客户演示和小规模交付。
2. 企业内网部署：适合涉密空间、工业现场、政府园区和私有项目。
3. 私有云部署：适合多团队、多项目、多地点协作。
4. 平台集成部署：可对接企业门户、OA、项目管理、对象存储、数字孪生平台和三维资产库。
